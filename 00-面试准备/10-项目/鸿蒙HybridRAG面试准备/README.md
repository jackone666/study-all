# 鸿蒙 Hybrid RAG 面试准备包

这套材料围绕简历中的“面向鸿蒙开发者的企业级 Hybrid RAG 智能问答系统”准备，分为三份：
1. `01-项目深挖学习文档.md`
   - 用来系统复盘项目背景、架构、数据流、关键设计和最佳实践补充。
   - 面试前先读这份，保证你能从全局讲清楚。
2. `02-高频面试问题库.md`
   - 按能力域整理可能被问的问题。
   - 包含追问方向和回答要点。
3. `03-回答稿与追问拆解.md`
   - 提供可直接背诵/转述的回答稿。
   - 重点覆盖自我介绍、项目介绍、架构深挖、缓存、去重、幻觉控制和反思题。

## 使用建议
- 第一遍：通读 `01-项目深挖学习文档.md`，把项目主线记住。
- 第二遍：用 `02-高频面试问题库.md` 自测，遇到答不上来的题回到学习文档补。
- 第三遍：背 `03-回答稿与追问拆解.md` 里的核心答案，但面试时不要逐字背，要用自己的话讲。

## 口径原则
- 这是 **Hybrid RAG / Workflow 系统**，不是自治 Agent 系统。
- 已在源码中能找到的能力，按“已实现”讲。
- 没有直接源码实现的能力，按“生产最佳实践/设计方案”讲，并说明为什么这么做。
- 面试官深挖时，优先讲数据流、边界条件、降级策略和 trade-off。
